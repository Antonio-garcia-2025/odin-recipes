# Odin Recipes

## Descripción

Odin Recipes es un proyecto de ejercicio creado para practicar HTML y CSS. Está diseñado como una pequeña aplicación web estática que muestra recetas de cocina, con foco en la estructura semántica, diseño visual y organización responsiva.

## Objetivo del proyecto

- Aprender y practicar conceptos básicos de HTML.
- Fortalecer habilidades de maquetación con CSS.
- Construir una interfaz simple y ordenada para presentar recetas.
- Usar buenas prácticas de accesibilidad y diseño responsivo.

## Contenido del repositorio

- `readme.md`: Documentación del proyecto.
- `index.html` (si existe): Página principal con las recetas.
- `styles.css` (si existe): Estilos para diseño y presentación.
- Imágenes u otros recursos (si se agregan): Apoyo visual para las recetas.


## Instalación y uso

1. Clona el repositorio:

   ```bash
   git clone https://github.com/antonio-garcia-2025/odin-recipes.git
   ```

2. Abre el proyecto en tu editor favorito.
3. Si existe un archivo `index.html`, ábrelo en el navegador para ver la página.
4. Modifica el contenido de HTML y CSS para agregar nuevas recetas y mejorar el estilo.

## Qué aprenderás

- Estructura de páginas web con etiquetas HTML (`header`, `main`, `article`, `section`, `footer`).
- Uso de clases e identificadores para aplicar estilos CSS.
- Diseño adaptable con `flexbox` o `grid`.
- Tipos de fuentes, colores y espaciado para mejorar la experiencia visual.

## Ideas para continuar

- Agregar más recetas con ingredientes y pasos.
- Incluir una sección de búsqueda o filtros.
- Añadir imágenes para cada receta.
- Implementar un menú de navegación.
- Hacer el diseño responsive para móviles y tablets.
